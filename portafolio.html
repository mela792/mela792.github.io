Proyecto 01

Aplicación web de gestión

Desarrollé una aplicación web completa para una empresa llamada Poética, en la que implementé un sistema de autenticación de usuarios con registro e inicio de sesión, así como una pasarela de pago integrada mediante PayPal para gestionar servicios premium de forma segura. Además, diseñé y desarrollé un panel de administración avanzado (SuperAdmin) que permite la gestión eficiente de usuarios, control de pagos y administración general de la plataforma. Todo el proyecto fue construido utilizando tecnologías clave del desarrollo web como HTML, CSS y JavaScript para el frontend, PHP para la lógica del servidor y MySQL para la gestión de la base de datos, demostrando mi capacidad para crear soluciones web completas, funcionales y orientadas a entornos reales.

Proyecto 02

TaskFlow | Gestión visual de tareas

En la web se muestra una aplicación sencilla y visual de gestión de tareas, con una interfaz limpia y fácil de entender. En la parte superior aparece una presentación del proyecto y, justo debajo, un formulario donde el usuario puede escribir una nueva tarea y añadirla a la lista con un solo clic. Más abajo se visualizan las tareas pendientes en formato de tarjetas, lo que permite ver de forma ordenada cada elemento registrado. Cada tarea incluye un selector para marcarla como completada, cambiando su apariencia para que se distinga claramente de las que siguen pendientes, y también un botón para eliminarla si ya no es necesaria. Además, la página incorpora mensajes de estado que informan al usuario cuando las tareas se están cargando, guardando, actualizando o eliminando, haciendo que la experiencia resulte más clara e interactiva. En conjunto, la web transmite la idea de una herramienta práctica para organizar actividades diarias, mostrando de manera directa cómo se pueden gestionar tareas desde una interfaz moderna, intuitiva y funcional.

Proyecto 03

Nova | Web personal con modo oscuro

Esta página web personal está diseñada para mostrar una estructura moderna y profesional con cabecera, navegación, contenido principal y pie de página, pero en este caso el aspecto más importante que se quiere destacar es el funcionamiento del modo oscuro. Desde el botón ubicado en el header, el usuario puede cambiar la apariencia completa del sitio, haciendo que el fondo y la gama de colores se adapten automáticamente para ofrecer una visualización diferente. Este efecto se consigue cuando JavaScript modifica el atributo del cuerpo de la página con la instrucción body.setAttribute("data-theme", theme);, y a partir de ese cambio el CSS aplica los nuevos colores definidos para el tema oscuro, entre ellos el fondo principal con la variable --bg: #0f172a;. De esta forma, la web no solo muestra una presentación personal clara y ordenada, sino también una funcionalidad visual dinámica que permite alternar entre modo claro y oscuro de manera sencilla.

Proyecto 04

WCAG | Accesibilidad web aplicada

Esta página web está diseñada para mostrar de forma práctica y visual distintas técnicas que permiten satisfacer los requisitos definidos en las WCAG, explicando no solo su importancia teórica, sino también su aplicación real dentro de una interfaz. A través de una estructura semántica clara, una navegación ordenada, un enlace para saltar al contenido principal, un contraste adecuado, estados de foco visibles y un formulario con etiquetas correctamente asociadas, la web demuestra cómo una página puede ser más accesible, comprensible y fácil de usar para todo tipo de personas. Además, incorpora un botón de contraste alto que permite modificar la visualización para mejorar la legibilidad, reforzando así la idea de que la accesibilidad no es solo una norma técnica, sino una parte esencial del diseño web moderno orientado a la inclusión y a una mejor experiencia de usuario.
